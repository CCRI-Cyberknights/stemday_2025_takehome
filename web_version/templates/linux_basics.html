üìÅ Example: Opening a Challenge Folder

  4. Run: cd 01_base64 (or whatever the folder is called).

üîß Common Linux Commands You'll Use in the CTF

üìÑ Reading & Inspecting Files

Print the full contents of a file.

Scroll through a long file. Press q to exit.

Show the first few lines (you can use head -n 20).

Show the last 10 lines (or use tail -f to follow updates).

üîç Searching Through Files

Search ignoring case (-i) and show line numbers (-n).

üóÇ Working With Compressed Files

üîé Inspecting Binary Files

Extract readable text from a binary. Used in Stego, Hex-Hunt, and Forensics challenges.

View a file in hex + ASCII.

üîê Encoding & Decoding

Encode a file into Base64.

üìú Running Scripts
